# Repo Overview

## What This Repository Is

This repository hosts a single-page personal portfolio website for Pariram S. The website is static and uses plain HTML, CSS, and JavaScript files with no build pipeline.

## Tech Summary

- HTML: `index.html`
- CSS: `assets/css/main.css`, `assets/css/fontawesome-all.min.css`
- JavaScript: jQuery and template utility scripts in `assets/js/`
- Icons: Font Awesome (local CSS and CDN stylesheet)
- Fonts: Google Fonts loaded from CSS (`Source Sans Pro`, `Raleway`)

## Runtime and Deployment Model

- Local run option 1: open `index.html` directly in a browser.
- Local run option 2: serve files through any static server.
- Deployment model: static hosting (any platform that serves HTML/CSS/JS/image files).

## Top-Level Repository Map

- `index.html`: main page markup and all portfolio content sections.
- `assets/`: styling and behavior scripts.
: `assets/css/` contains global and theme CSS.
: `assets/js/` contains jQuery and template behavior scripts.
- `images/`: image assets used by the page.
- `README.md`: root-level short project readme.
- `md/`: contributor-focused repository documentation (this folder).
